## Установка
1. Клонировать репозиторий
   ```bash
   git clone https://github.com/RealAnski1/Hakaton-AI.git
   ```
2. Устанавливать зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

## Использование
### Если админ то:
1. Напишите "/start"
2. Выберите места Красноярска
3. посмотрите у мест отзывы (если есть)
4. Выберите место которое понравилось
5. Выберите добавить в избранное или посетили
6. Если вы посетили то напишите отзыв
7. Теперь можете посмотреть места где были и избранные
8. Если вы в избранном и вам больше не нравится это место то можно удалить его из избранного

## База данных

По умолчанию используется SQLite. Файл базы данных: `db.db`

## Подробности для добавления новых мест
### Типы заведений (type):
1. Отель
2. Кафе
3. Ресторан
4. Продуктовый
5. Фирменный магазин (Опционально для города)

### Имя (name):
- Неопционально, может быть на любом языке.

### Рейтинг (rate):
1.  Низший рейтинг заведения
2.  Плохое заведение
3.  Более менее
4.  Неплохое заведение
5.  Высший рейтинг заведения
